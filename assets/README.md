# Dossier Assets

Ce dossier contient les ressources graphiques de l'application.

## Logo de l'en-tête

**Fichier requis :** `prevenir.png`

**Spécifications :**
- Format : PNG avec transparence (recommandé)
- Hauteur : Sera automatiquement redimensionné à 60px de hauteur
- Largeur : Proportionnelle (pas de limite)
- Emplacement : `/assets/prevenir.png`

**Note :** Si le logo n'est pas trouvé, l'application affichera "Pointage" en texte comme fallback.

## Comment ajouter le logo

1. Placez votre fichier `prevenir.png` dans ce dossier
2. Relancez l'application
3. Le logo apparaîtra automatiquement dans l'en-tête
